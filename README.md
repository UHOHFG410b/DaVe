# DaVe Abschlussbericht – Digitale Ergänzung

## Description
Dieses Repository dient als interaktive Erweiterung zum statischen Abschlussbericht des Projekts **DAVE**. Da die Diagramme der Kostenanalysen (AP 2) und der Datenflussanalyse (AP 3) sehr groß und verschachtelt sind, bietet dieses Repo die notwendige Klickbarkeit und Detailtiefe, die im PDF-Anhang des Berichts technisch nicht darstellbar sind. Außerdem wird so die Nutzung und Weiterentwicklung der Rohartefakte ermöglicht.

## Visuals
* **AP 2:** Detailansichten und Tabellen der Kostenanalyse.
* **AP 3:** Interaktive, verschachtelte Datenfluss-Diagramme (BPMN).

## Access

Die Artefakte sind über zwei komplementäre Zugangswege verfügbar:

### 1. GitHub Pages (empfohlen für Analyse)

Interaktive Darstellung der Diagramme:

- Visualisierung der Kostentreiber und Datenflüsse  
- Navigation durch Prozesse und Akteure  
- Bessere Lesbarkeit komplexer Zusammenhänge als im statischen Bericht

### 2. GitHub Repository (für Weiterverarbeitung)

Download der Rohartefakte und Nutzung der Dateien:

👉 Zugriff: https://github.com/UHOHFG410b/DaVe/

---

### Nutzung nach Arbeitspaketen

* **AP 2:** Keine Installation oder gesonderte Programme notwendig.  
    1. Repository als ZIP herunterladen oder via Git klonen.  
    2. Die `.html`- oder `.svg`-Dateien direkt im Webbrowser öffnen.  

* **AP 3:** Die BPMN-Dateien liegen im `.bpmn`-Format vor. Für die optimale Ansicht empfehlen wir:  
    1. **Signavio (Empfohlen):** Bietet die beste, farbige Darstellung und volle Interaktivität. Hierfür ist ein (kostenloses) Konto bei [Signavio](https://www.signavio.com/) erforderlich.  
    2. **Alternative Viewer:** Weniger farbenreich, aber ohne Registrierung nutzbar, sind Tools wie der [Camunda Modeler](https://camunda.com/download/modeler/) oder webbasierte Viewer wie [bpmn.io](https://demo.bpmn.io/).
